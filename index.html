Scrollon

Scrollonの魅力

処理内容はクラスの付与だけ。とてもシンプルです。
そのため、アニメーションの実装は全てCSSに任せることができ、少しの工夫で様々な出現方法を実現できます。
このデモページでは、その例をいくつか紹介しようと思います。

* 当デモページでは便宜上、Scrollonの呼び出しを以下のようにしていますが、本来はもっとシンプルです。（使用方法については README をご覧ください。）

body閉じタグ直前にて 

<script>
    new Scrollon('.item1, .item2, .item3, .item4');
</script>

単純なフェードイン

CSS 

.item1{
    opacity: 0;
}
.item1.scon{
    opacity: 1;
    transition: all 2s;
}

下からフェードイン

CSS 

.item2{
    position: relative;
    top: 30px;
    opacity: 0;
}
.item2.scon{
    top: 0;
    opacity: 1;
    transition: all 2s;
}

左からフェードイン

CSS 

.item3{
    position: relative;
    right: 30px;
    opacity: 0;
}
.item3.scon{
    right: 0;
    opacity: 1;
    transition: all 2s;
}

リストを順に下からフェードイン

CSS 

@keyframes fade_list {
  0% {
    top: 30px;
    opacity: 0;
  }

  100% {
    top: 0;
    opacity: 1;
  }
}

.item4 li{
    position: relative;
    top: 30px;
    opacity: 0;
}

.item4.scon li:nth-child(1){
    animation: fade_list 2s ease 0s 1 normal both running;
}
.item4.scon li:nth-child(2){
    animation: fade_list 2s ease 0.3s 1 normal both running;
}
.item4.scon li:nth-child(3){
    animation: fade_list 2s ease 0.6s 1 normal both running;
}
.item4.scon li:nth-child(4){
    animation: fade_list 2s ease 0.9s 1 normal both running;
}
.item4.scon li:nth-child(5){
    animation: fade_list 2s ease 1.2s 1 normal both running;
}
.item4.scon li:nth-child(6){
    animation: fade_list 2s ease 1.5s 1 normal both running;
}
.item4.scon li:nth-child(7){
    animation: fade_list 2s ease 1.8s 1 normal both running;
}
.item4.scon li:nth-child(8){
    animation: fade_list 2s ease 2.1s 1 normal both running;
}

@keyframesアニメーションを利用すれば、親にクラスを付与するだけで子要素を順番にフェードさせることも簡単です。
上記ではただのCSSでの記述なので少し手間がかかりますが、Sassを利用すれば関数でループ処理できるので手軽に実装できます。
